# 🚗 Parking Space Detection using Real-Time Camera Feeds

## 📌 Project Overview
This project detects parking space occupancy in real time using **YOLOv8** and **OpenCV**. The system processes live or recorded video feeds, checks predefined parking zones, and identifies whether each slot is occupied or free.

## ⚙️ Features
- Real-time car detection using YOLOv8  
- Polygon-based parking slot tracking (via `carPark.json`)  
- Visual indicators for occupied vs. free spaces  
- Works on recorded videos or live camera input  

## 🛠️ Tech Stack
- Python  
- OpenCV  
- YOLOv8 (Ultralytics)  
- cvzone  
- NumPy
